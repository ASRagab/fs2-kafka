Overview

Functional backpressured streams for consuming and producing Kafka records. Exposes a minimalistic interface, while taking care of common functionality: batch consuming and producing records, batched offset commits, offset commit recovery, and administration, while also simplifying client configuration.

Documentation is kept up-to-date with new releases, currently documenting v0.18.0 on Scala 2.12.

Getting Started

To get started with sbt, simply add the following line to your build.sbt file.

libraryDependencies += "com.ovoenergy" %% "fs2-kafka" % "0.18.0"

Published for Scala 2.11 and 2.12. For changes, refer to the release notes.

Backwards binary-compatibility for the library is guaranteed between patch versions.
For example, 0.18.x is backwards binary-compatible with 0.18.y for any x > y.

Remember to enable partial unification by adding the following line to build.sbt.

scalacOptions += "-Ypartial-unification"

Dependencies

Has a minimal set of dependencies:

  • FS2 v1.0.2 (Documentation, GitHub), and
  • Apache Kafka Client v2.0.1 (Documentation, GitHub).

Inspiration

Library is heavily inspired by ideas from Alpakka Kafka.
